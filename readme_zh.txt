Doc-Copilot - 文档辅助工具

概述:
Doc-Copilot是一个浏览器扩展，帮助用户更高效地创建和管理文档。它提供基于AI的建议和文档创建自动化功能。

功能:
- 基于AI的文档建议
- 实时协作支持
- 跨平台文档管理
- 可自定义模板

技术栈:
- 前端: React + TypeScript
- 构建工具: Vite
- 扩展框架: Chrome Extension API

安装:
1. 克隆本仓库
2. 运行 `npm install`
3. 运行 `npm run dev` 进行开发
4. 在Chrome开发者模式下加载此扩展

使用:
1. 点击扩展图标打开弹出窗口
2. 选择文档模板或从头开始创建
3. 使用AI建议增强您的文档
4. 保存或导出您的工作
