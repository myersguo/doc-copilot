# Privacy Policy 隐私政策

## Data Collection 数据收集
This extension collects the following user-provided data:
- API Key (stored securely in chrome.storage.sync)
- URL patterns where the extension should activate

本扩展收集以下用户提供的数据：
- API密钥（安全存储在chrome.storage.sync中）
- 扩展应激活的URL模式

## Data Storage 数据存储
All user configuration data is stored locally in your browser using chrome.storage.sync, which synchronizes across devices when signed into Chrome.

所有用户配置数据都使用chrome.storage.sync本地存储在您的浏览器中，当登录Chrome时会在设备间同步。

## Data Usage 数据使用
- API Key is used exclusively to authenticate with the AI service for text completion functionality
- URL patterns are used to determine where the extension should activate

- API密钥仅用于验证AI服务的文本补全功能
- URL模式用于确定扩展应在何处激活

## Data Sharing 数据共享
We do not share any user data with third parties. The extension operates entirely within your browser.

我们不会与第三方共享任何用户数据。该扩展完全在您的浏览器中运行。

## User Rights 用户权利
You can view, modify, or delete your data at any time through the extension's options page. To completely remove all extension data, uninstall the extension.

您可以通过扩展的选项页面随时查看、修改或删除您的数据。要完全删除所有扩展数据，请卸载扩展。
